## General info

It is my training project of italian pizza restaurant website. Main goal was for me to finally use learnt skills about webpack, react and bootstrap frameworks as well as get better with using github. Main styling were used from bootstrap with some custom additions in Scss and animations with Css Transition Group. It's a static page, without any backend needed.

## Technologies

Project is created with:

- React version: 16.13.1
- Bootstrap version: 4.5.1
- Webpack version: 4.44.1
- jQuery version: 3.5.1
- React Router Dom version: 5.2.0
- Scss

## Setup

To run this project, install it locally using npm:

```
$ cd ../Lastravaganza-Project
$ npm install
$ npm start
```
